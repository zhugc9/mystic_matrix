# Mystic Matrix 售卖打包说明

## 目标
- 客户下载后双击即可使用
- 客户只需改提示词、算命、管理历史
- 尽量提高源码可见门槛

## 推荐交付形态
1. 运行 `一键打包EXE.bat`
2. 生成 `dist/MysticMatrix-Portable-*.exe`
3. 交付给客户：
   - `启动赛博玄学.bat`
   - `dist/MysticMatrix-Portable-*.exe`

## 客户操作流程
1. 双击 `启动赛博玄学.bat`
2. 在主页配置 API Key
3. 打开 `提示词中心` 修改提示词
4. 进入各模块进行占卜
5. 在 `命运回响` 管理历史记录与聚合报告

## 源码保护现实
- EXE 可提高门槛（相对网页版）
- 无法做到 100% 不可逆向
- 如需更高保护，可结合代码混淆与授权机制
