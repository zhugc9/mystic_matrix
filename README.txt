# 赛博玄学矩阵 (Mystic Matrix) - 最终交付版

## 🔮 关于产品
这是一个集成了塔罗、易经、手相、面相与星盘的“赛博玄学”应用。
它融合了古老的东方智慧与现代的赛博朋克美学，旨在通过“仪式感”唤醒使用者的潜意识。

## 🚀 如何运行
**双击运行 `OneClickStart.bat` 即可进入矩阵。**

- **推荐环境**：安装了 Python 的 Windows 电脑（可开启本地服务器，摄像头支持更好）。
- **极简模式**：无 Python 环境也可直接运行，但部分浏览器可能会限制摄像头的本地访问。

## ⚙️ 首次配置
1.  进入应用后，请点击主页右下角的 **“系统设置”**。
2.  输入您的 LLM API Key（推荐 DeepSeek / OpenAI 等兼容接口）。
    - 接口地址默认：`https://api.deepseek.com/chat/completions`
    - 模型名称默认：`deepseek-reasoner`
3.  点击 **“摄像头权限”** 按钮，授予浏览器访问权限（用于手相/面相/易经手势）。

## 📁 目录结构
- `index.html`: 矩阵入口（命运回响）
- `common/`: 核心逻辑库
- `tarot/` `iching/` ...: 各大功能模块
- `OneClickStart.bat`: 一键启动脚本
